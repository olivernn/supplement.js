Supplement.js

JavaScript
Extras

A small collection of utility functions to make working with JavaScript that much sweeter and more expressive.

Responsibly adds extensions to built in types.

Usage

Include the supplement.js source before any JavaScript in which you wish to use the extras.

Supplement.js has a single dependency on its sister library Augment.js which should be included before supplement.js.

Methods

  • Array.wrap - Wraps its argument in an array, unless it is already an array.

  • Array.range - Creates flexible lists of numbers - handy for forEach and map loops.

  • Array.toArray - Converts an array-like object into a real array - useful for converting the argument's object.

  • Array.prototype.detect - Returns the first element for which the function returns true, then stops iterating.

  • Object.values - Returns an array of all enumerable property values directly belonging to the given object.

  • Object.provide - Ensures the existence of the object structure provided by the arguments. It checks whether each property form the arguments exists, and if not, initialises it.

  • Function.prototype.singleUse - Returns a function that can only be used once.

  • Function.prototype.throttle - Returns a function that is throttled to the rate provided as an argument.

  • Function.prototype.debounce - Returns a function whose execution is delayed for a defined amount of time.

  • Function.prototype.curry - Returns a function that has had some of its arguments preset.

  • Number.prototype.times - Executes the passed function a given number of times.
